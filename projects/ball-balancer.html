About This Game

A physics-based ball balancing simulation built with modern C++ and compiled to WebAssembly using Emscripten. This demonstrates running native C++ applications directly in your browser with near-native performance.

How to Play

  • Mouse Movement: Control the platform tilt
  • Click and Drag: Adjust platform angle for precise control
  • ImGui Controls: Use the on-screen GUI to adjust physics parameters in real-time

Technical Stack

Technologies: C++, WebAssembly, OpenGL/WebGL, ImGui, Physics Simulation
